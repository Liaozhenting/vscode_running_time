# vscode_running_time
Vscode 运行时间统计


## 设置
在 start.bat找到
```
start /d "D:\Program Files\Microsoft VS Code" Code.exe
```
把`"D:\Program Files\Microsoft VS Code"`替换为本机vscode的安装路径；

## 初始化时间(默认是初始的，不用管这条)
./last-launch,./total-running-time文件的内容都写成0。

./last-launch
```
0
```

./total-running-time
```
0
```
